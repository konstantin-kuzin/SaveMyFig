# История изменений

## [2.0.0] - 2025-10-31

### Добавлено
- Полностью новый графический интерфейс на Electron
- Автоматическая установка Node.js и зависимостей при первом запуске
- Удобная настройка параметров Figma через GUI
- Мониторинг процесса бэкапа в реальном времени
- Просмотр статистики и базы данных SQLite
- Встроенные инструменты диагностики системы
- Поддержка нескольких аккаунтов Figma
- Валидация конфигурации с подсказками
- Автоматическая генерация files.json
- Система уведомлений о завершении операций
- Темная тема интерфейса
- Адаптивный дизайн для разных размеров экранов

### Изменено
- Обновлена архитектура проекта с разделением на CLI и GUI
- Улучшена обработка ошибок с детализированными сообщениями
- Оптимизирована работа с базой данных
- Улучшена документация с подробными руководствами

### Исправлено
- Проблемы с авторизацией через Auth Cookie
- Ошибки при обработке больших файлов
- Проблемы с повторными попытками загрузки
- Несовместимость с новыми версиями Node.js

## [1.2.1] - 2025-03-15

### Исправлено
- Ошибка с неправильной обработкой дат в базе данных
- Проблемы с кодировкой имен файлов на Windows

## [1.2.0] - 2025-02-28

### Добавлено
- Поддержка Project IDs в дополнение к Team IDs
- Скрипт генерации отчетов по базе данных
- Расширенное логирование операций
- Проверка целостности загруженных файлов

### Изменено
- Улучшена обработка ошибок с более подробными сообщениями
- Оптимизирована работа с базой данных SQLite
- Обновлена документация с примерами конфигурации

### Исправлено
- Проблемы с таймаутами при медленном интернет-соединении
- Ошибки при обработке файлов с нестандартными символами в именах

## [1.1.2] - 2024-12-10

### Исправлено
- Ошибка с неправильным расчетом времени следующей попытки
- Проблемы с авторизацией при использовании нескольких аккаунтов

## [1.1.1] - 2024-11-25

## [1.1.0] - 2024-11-15

- Автоматическая ротация аккаунтов при достижении лимитов

### Изменено

### Исправлено

## [1.0.0] - 2024-10-15

- Конфигурация через .env файл
- Скрипты для запуска различных операций
- Подробная документация с примерами
